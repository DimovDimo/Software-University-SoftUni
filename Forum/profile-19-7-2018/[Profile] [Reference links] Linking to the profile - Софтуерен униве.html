Активно записване за предстоящи модули от септември 2018

През сезон септември 2018 в СофтУни ще стартират 7 паралелни учебни модула. 

- Записването за Tech Module е активно от ТУК.

- Записването за модул C# Fundamentals е активно ТУК.

- Записването за модул Java DB Fundamentals е активно ТУК.

- Записването за модул JS Core е активно ТУК.

- Записването за модул C# Web е активно ТУК.

- Записването за модул PHP Web е активно ТУК.

- Записването за модул Front-End Development е активно ТУК.

Записването се осъществява след успешно извършено плащане. Срокът за плащане е 6 септември 2018 година (включително).

Лекциите за професионалните модули ще се провеждат предимно вечер, а упражненията - предимно следобед, като ще бъдат записвани и достъпни в страницата на курса.
